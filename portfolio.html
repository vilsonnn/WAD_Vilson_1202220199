Notebook Project 1: Predicting Promotional Campaign Success ▼

Proyek ini bertujuan untuk memprediksi kesuksesan kampanye promosi menggunakan Machine Learning. Berfokus pada pemrosesan data historis kampanye pemasaran dan variabel terkait seperti demografi dan perilaku konsumen, proyek ini menghasilkan model prediktif untuk meningkatkan efisiensi kampanye di masa depan.

Project Sentinel 1: Deteksi Komentar Bullying ▼

Proyek ini mengembangkan sistem deteksi otomatis untuk komentar bullying di media sosial menggunakan teknik NLP dan Machine Learning. Model dilatih pada dataset yang berisi komentar berbahaya untuk mendeteksi dan mengklasifikasikan komentar bullying dengan tingkat akurasi yang tinggi.
